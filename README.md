# Iris-Flower-Classification
Machine learning model to classify iris flowers into setosa, versicolor, and virginica using scikit-learn.


# 🌸 Iris Flower Classification

## 📌 Objective
The objective of this project is to build a machine learning model that can classify iris flowers into three species:
- Setosa
- Versicolor
- Virginica

based on their measurements such as sepal length, sepal width, petal length, and petal width.

---

## 📊 Dataset
The dataset is taken from the Scikit-learn library (Iris dataset), which contains 150 samples of iris flowers.

---

## ⚙️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🚀 Steps Performed
1. Loaded the dataset using Scikit-learn  
2. Converted data into a DataFrame  
3. Performed data visualization using graphs  
4. Split the dataset into training and testing sets  
5. Trained the model using Logistic Regression  
6. Predicted the species of iris flowers  
7. Evaluated the model using accuracy score  

---

## 🤖 Model Used
- Logistic Regression

---

## 📈 Result
The model successfully classified iris flowers with high accuracy (around 90–100%).

---

## 📷 Output
The notebook includes visualizations such as:
- Pairplot
- Scatter Plot

---

## 📌 Conclusion
The machine learning model is able to accurately classify iris flowers based on their features.
